Produits et licences SQL Anywhere

SQL Anywhere est disponible sous forme de "base package" (package de base), complété par des options et des licences de déploiement. Pour commencer, vous choisissez le base package le mieux adapté à vos besoins. Deux modèles de licence sont proposés : (a) par serveur et poste ; (b) par processeur.

Dans le modèle par serveur et poste, SQL Anywhere est disponible sous forme de base package pour 5 utilisateurs et pour 10 utilisateurs. Il inclut un serveur réseau SQL Anywhere et le nombre de licences correspondant au nombre d'utilisateurs choisi. Chacun des utilisateurs dispose d'une licence de serveur personnel SQL Anywhe re, de la fonctionnalité de synchronisation ainsi que de QAnywhere Messaging et d'Ultralite avec la synchronisation.

Vous pouvez également opter pour le modèle de licence de serveur SQL Anywhere par processeur. Ce package fournit une licence du serveur réseau SQL Anywhere pour une machine et pour le nombre de processeurs choisi. Le nombre d'utilisateurs autorisés à se connecter au serveur avec un licence par poste en réseau est illimité. C'est donc la solution idéale pour les déploiements Web et client/s erveur de grande envergure, ou lorsqu'un système de licence par processeur est préférable pour éviter de devoir comptabiliser les postes ou les utilisateurs.

Le base package SQL Anywhere inclut également des licences pour un certain nombre d'outils de conception et de gestion tels que Sybase Central, le Modèle physique de données de PowerAMC et Data Window.NET pour SQL Anywhere.

Licences de déploiement
