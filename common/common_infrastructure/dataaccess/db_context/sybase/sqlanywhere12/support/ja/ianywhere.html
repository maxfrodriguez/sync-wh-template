制限事項・注意事項
製品をお使いになる前に、制限事項・注意事項についてこちらをご参照ください。

SQL Anywhere ユーザー登録
追加ライセンスのご購入の際には、製品のユーザー登録時に発行されたユーザー ID が必要となります。パッケージ同梱の登録用紙にて、必ず製品ユーザー登録をしてください。

EBF (Express Bug Fix、不具合修正パッチ)
SQL Anywhereは、次期バージョンアップまでの間、不具合修正パッチとして EBF (Express Bug Fix) を提供しています。提供サイクルは不定ですが、サポート OS における変更 (Windows でのサービスパック等) による修正なども EBF で提供しています。
EBF の使用につきましては、有償サポート契約が必要となります。

更新のチェック
お客様がお使いの製品以降提供されている EBF (Express Bug Fix、 不具合修正パッチ) 等については、こちらよりご確認いただけます。

SQL Anywhere の製品とライセンス

基本パッケージおよび追加ライセンス

SQL Anywhere では、数種ある基本パッケージの中から1つを選択、購入いただき、基本パッケージで許諾されている各コンポーネントのライセンス数を超えるものについて、追加ライセンスをご購入いただきます。
基本パッケージには、(a) マシンの台数ベースのサーバー/シートライセンスモデルと、(b) 物理 CPU 数ベースの CPU ライセンスモデルがあります。
各基本パッケージで許諾されているライセンス他、SQL Anywhere のライセンスについては、こちらをご参照ください。

開発者のサポート

テクニカル・サポート
アイエニウェアでは、お客様が世界最高水準のソリューションを迅速かつコスト効率よく開発・導入していただけるよう、正確で時宜を得た技術支援のメニューを用意しています。

SQL Anywhere デベロッパー・コミュニティ
クライアント・サーバーアプリケーション、デスクトップアプリケーション、モバイルアプリケーション、Web アプリケーション、リモートオフィス・ソリューションの開発・展開に役立つテクニカル情報を提供しています。世界各地のエキスパートがこの開発者コミュニティに貢献しているため、ここに集約管理された、第一線の環境におけるアプリケーション開発に関するすべての情報を得ることができます。

ニュースグループ
アイエニウェアのニュースグループは、誰でもアクセス・参加でき、 開発に関する問題について開発者コミュニティと議論することができます。こちらのニュースグループは、アイエニウェア・ソリューションズの日本の技術部隊が支援しています。

DocCommentXchange
SQL Anywhere のマニュアルの個々の記載内容に関して寄せられたコメントを参照したり、自らコメントを入力することができます。

その他製品情報
アイエニウェアでは、現場の第一線で使用されるアプリケーションの信頼性、セキュリティ、可用性を、本社で使用されるアプリケーション同様に高めるソリューションを提供しています。モバイル管理とセキュリティー、モバイル・ミドルウェアと同期において業界をリードする当社の製品について、Web サイトを参照してください。

パートナープログラム
SQL Anywhere を使用したアプリケーションの開発・販売をされる企業向けプログラムです。競争に打ち勝つための業界をリードするテクノロジーに基づいた付加価値の高いサービスを提供します。パートナーの皆さまがビジネス・チャンスを最大限に生かし、両社にとっての成功となるよう支援します。

プロフェッショナル・サービス
お客様の個別要望に基づく具体的なアドバイスや、設計・開発案件の内容に合わせて SQL Anywhere 製品をより有効に活用していただくための最適なアドバイス・技術支援を提供するサービスのご案内です。

  • SQL Anywhere の最新バージョンへのアップグレードの計画と実施
  • SQL Anywhere のアーキテクチャーをご理解いただくための知識伝授
  • 導入環境のパフォーマンスと全体的な品質の確保
  • Mobile Link 同期ソリューションの設計・開発の支援
  • SQL Anywhere の高可用性の設定と実装

トレーニング・サービス
SQL Anywhere による独自アプリケーションの開発・展開を支援するため、SQL Anywhere のトレーニングを提供します。トレーニング内容が基礎レベルであるか上級レベルであるかを問わず、標準の教育コースとお客様のご要望に合わせたトレーニング・プログラムをご用意しています。トレーニング・プログラムでは、データベースの機能、データベース間データ同期技術、モバイル機器への配布・展開などのトピックを網羅しています。
