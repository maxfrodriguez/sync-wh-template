Produkte und Lizenzierung für SQL Anywhere

SQL Anywhere wird in mehreren Basispaketen und mit zusätzlichen Deployment-Lizenzen und Optionen angeboten. Zunächst wählen Sie das Basispaket, das Ihren Anforderungen am besten entspricht. Basispakete sind in zwei Lizenzmodellen verfügbar: (a) Server und Seats sowie (b) pro Prozessor.

Beim Server- und Seats-Modell wird SQL Anywhere als Basispaket für 5 Benutzer und 10 Benutzer angeboten, das jeweils einen SQL Anywhere-Netzwerkserver und die festgelegte Anzahl von Lizenzen umfasst. Jeder dieser Benutzer hat eine Lizenz für einen Networked Seat und damit Clientzugriff auf den SQL Anywhere-Server. Die Benutzer verfügen außerdem über eine Lizenz für den SQL Anywhere Personal Server mit Synchronisation und QAnywhere Messaging sowie UltraLite mit Synchronisation.

Sie können sich als Alternative aber auch für eine Pro-Prozessor-Lizenz des SQL Anywhere-Servers entscheiden. Bei diesem Paket erhalten Sie eine Lizenz für den SQL Anywhere-Netzwerkserver auf einem Rechner für die Anzahl der gekauften Pro-Prozessor-Lizenzen. In diesem Fall besteht keine Begrenzung der Anzahl von Benutzern, die sich als Networked Seats mit dem Server verbinden können. Dieses Paket ist daher ideal für größere Client/Server- und webbasierte Deployments oder wenn Sie eine Lizenz pro Prozessor bevorzugen, anstatt die Anzahl der Benutzer oder Seats zählen zu müssen.

SQL Anywhere-Basispakete enthalten darüber hinaus Lizenzen für eine Reihe von Design- und Verwaltungstools wie Sybase Central, PowerDesigner Physical Data Model und Data Window.NET.

Deployment-Lizenzen

Lizenzen für weitere Benutzer oder für zusätzliche Komponenten, wie z.B. zusätzliche Network Seats (Clients), Netzwerkserver, Personal Server, Synchronisation, QAnywhere Messaging und UltraLite, können nachträglich erworben werden.
