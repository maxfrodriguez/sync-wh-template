SQL Anywhere 产品和授权

SQL Anywhere 以具有额外部署许可和选件的多个基础软件包形式提供。首先，选择最能满足您要求的基础软件包。基础软件包以两种授权模式提供：(a) 服务器和存取设备模式；(b) 基于处理器的模式。

在服务器和存取设备模式下，SQL Anywhere 以 5 用户和 10 用户基础软件包形式提供，其中包括 SQL Anywhere Network Server 以及所述数量的用户许可证。这些用户中的每一个都被授权作为网络存取设备，以向 SQL Anywhere Server 提供客户端访问。这些用户中的每一个还被授权使用具有同步的 SQL Anywhere Personal Server、QAnywhere Messaging 以及具有同步的 Ultralite。

客户也可选择 SQL Anywhere Server CPU 许可。此软件包授权 SQL Anywhere Network Server 在一台计算机上具有所购买数量的 CPU 许可。对可作为联网存取设备连接到服务器的用户的数量没有限制，这就使此软件包特别适合于较大型客户端/服务器和基于 Web 的部署，或者客户偏爱基于处理器的授权模式而不想要计数用户或存取设备的情况。

SQL Anywhere 基础软件包还包括授权很多设计和管理工具，其中包括 Sybase Central、PowerDesigner Physical Data Model 及 Data Window.NET for SQL Anywhere。

部署许可

需要授权更多的用户或额外的组件吗？没问题。可针对额外的网络存取设备（客户端）、网络服务器、个人服务器、同步、QAnywhere Messaging 和 Ultralite 进行授权。
